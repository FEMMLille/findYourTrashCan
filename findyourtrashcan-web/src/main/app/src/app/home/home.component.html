Soyez citoyen, chaque jours trouvez une poubelle proche de vous pour jeter vos déchets.

FindYourTrashcan™ est une application communautaire permettant de trouver une poubelle rapidement proche de vous. Rejoinez les citoyens proche de vous afin de contribuer à la propreté de votre environnement, réduisant la polution et améliorant votre vie quotidienne.
